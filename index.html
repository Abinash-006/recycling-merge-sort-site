Merge Sort for Waste Recycling Data

Sort categories like Plastic, Paper, Glass, Metal — efficient, stable, and perfect for merging collected batches.

Why Merge Sort?

Merge Sort is a divide-and-conquer stable sorting algorithm with O(n log n) time complexity. It's great for sorting recycling records where you may want to merge multiple collection batches (stable order keeps items of the same category grouped predictably).

Merge Sort — Steps (high level)

  1. Split the array in half until each piece has ≤ 1 item.
  2. Merge pairs of pieces back together in sorted order.
  3. Continue merging until you have a single sorted array.
MergeSort(A)
  if length(A) ≤ 1: return A
  mid ← floor(length(A)/2)
  left ← MergeSort(A[0..mid-1])
  right ← MergeSort(A[mid..end])
  return Merge(left, right)

Merge(left, right)
  result ← empty
  while left and right not empty:
    if left[0].key ≥ right[0].key:
      append left[0] to result; pop left[0]
    else:
      append right[0] to result; pop right[0]
  append remaining items
  return result

Above pseudocode sorts by weight descending (note ≥ in comparison). For ascending use < instead.
